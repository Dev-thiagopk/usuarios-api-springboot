# API REST de Usuários

Projeto desenvolvido em Java com Spring Boot

## Funcionalidades
- Criar usuário
- Listar usuários
- Buscar por ID
- Atualizar usuário
- Deletar usuário

##  Tecnologias
- Java
- Spring Boot
- H2 Database

## Como rodar
1. Rodar a classe principal (Application)
2. Acessar: http://localhost:8080/usuarios
